Запустить приложение
Сохраните launcher в папку office-net-monitor, затем двойной щелчок. Держите Terminal или окно команд открытым.
Mac — START.command
Mac — Network Monitor.app (zip)
Windows / PC — START.bat
Уже в папке проекта? Используйте START.command (Mac) или START.bat (PC) — скачивание не нужно.
Открыть этот сайт
Используйте http://127.0.0.1:5080/ — не файл из Finder.
Сканировать сеть
В списке устройств нажмите Scan Wi‑Fi + LAN. Подождите 1–2 мин.
Смотреть карту
Откройте Network map для Internet → gateway → devices.
Именовать устройства
Введите в столбец IT name (принтер, PC, комната и т.д.).
