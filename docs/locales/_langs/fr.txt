Démarrer l'application
Enregistrez le lanceur dans le dossier office-net-monitor, puis double-cliquez. Gardez Terminal ou la fenêtre de commande ouverte.
Mac — START.command
Mac — Network Monitor.app (zip)
Windows / PC — START.bat
Déjà dans le dossier du projet ? Utilisez START.command (Mac) ou START.bat (PC) — pas de téléchargement nécessaire.
Ouvrir ce site
Utilisez http://127.0.0.1:5080/ — pas un fichier depuis le Finder.
Scanner le réseau
Sur la liste des appareils, cliquez Scan Wi‑Fi + LAN. Attendez 1–2 min.
Voir la carte
Ouvrez Network map pour voir Internet → gateway → devices.

Ce parcours m'a appris les opérations de bout en bout : les personnes, le timing et une communication claire. Aujourd'hui, je me concentre sur les opérations et la coordination de projets — transformer un travail complexe en systèmes simples et reproductibles.
