Tutorial rápido
Seis pasos — configuración inicial
Iniciar la aplicación
Guarde el iniciador en la carpeta office-net-monitor y haga doble clic. Mantenga Terminal o la ventana de comandos abierta.
Mac — START.command
Mac — Network Monitor.app (zip)
Windows / PC — START.bat
¿Ya está en la carpeta del proyecto? Use START.command (Mac) o START.bat (PC) — no necesita descargar.
Abrir este sitio
Use http://127.0.0.1:5080/ — no un archivo del Finder.
Escanear la red
En la lista de dispositivos, haga clic en Scan Wi‑Fi + LAN. Espere 1–2 min.
Ver el mapa
Abra Network map para ver Internet → gateway → devices.
Nombrar dispositivos
Escriba en la columna IT name (impresora, PC, sala, etc.).
