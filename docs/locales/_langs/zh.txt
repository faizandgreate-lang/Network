启动应用
将启动器保存到 office-net-monitor 文件夹，然后双击。保持 Terminal 或命令窗口打开。
Mac — START.command
Mac — Network Monitor.app (zip)
Windows / PC — START.bat
已在项目文件夹中？使用 START.command (Mac) 或 START.bat (PC) — 无需下载。
打开此站点
使用 http://127.0.0.1:5080/ — 不要从 Finder 打开文件。
扫描网络
在设备列表上点击 Scan Wi‑Fi + LAN。等待 1–2 分钟。
查看地图
打开 Network map 查看 Internet → gateway → devices。
