Mulai aplikasi
Simpan launcher di folder office-net-monitor, lalu klik ganda. Biarkan Terminal atau jendela perintah terbuka.

Mac — Network Monitor.app (zip)
Windows / PC — START.bat
Sudah di folder proyek? Gunakan START.command (Mac) atau START.bat (PC) — tidak perlu unduh.
