Quick tutorial

Six steps — first time setup

01

Download & start

Download the full package, unzip it, then double-click START.command (Mac) or START.bat (Windows). Keep Terminal or the command window open.

Or download the whole repo as ZIP from GitHub (Code → Download ZIP), unzip, then run START in that folder.

Full app (recommended)

Mac — full download (.zip) Windows / PC — full download (.zip) GitHub — whole project (.zip)

Starter only

Mac — START.command only Windows — START.bat only

Mac app

Mac — Network Monitor.app (.zip)

Requires Python 3. First run installs dependencies automatically.

02

Open this site

Use http://127.0.0.1:5080/ — not a file from Finder.

03

Scan the network

On Device list, click Scan Wi‑Fi + LAN. Wait 1–2 min.

04

View the map

Open Network map to see Internet → gateway → devices.

05

Name devices

Type in the IT name column (printer, PC, room, etc.).

06

Download reports
